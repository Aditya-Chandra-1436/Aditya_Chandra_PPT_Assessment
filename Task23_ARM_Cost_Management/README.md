# Task 23 – Azure Resource Manager & Cost Management (Easy)

## Simple Definition
Azure Resource Manager (ARM) is used to organize and manage Azure resources, while Azure Cost Management helps track and analyze cloud spending.

## Objective
To understand resource grouping using ARM and view cost analysis using Azure Cost Management.

## Azure Services Used
- Azure Resource Manager
- Azure Cost Management

## Steps Performed
1. Created a resource group using Azure Resource Manager.
2. Viewed subscription cost analysis using Azure Cost Management.
3. Explored cost tracking and resource grouping concepts.

## Result
Successfully organized resources using ARM and viewed cost analysis in Azure.
