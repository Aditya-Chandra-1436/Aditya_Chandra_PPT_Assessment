# Task 17 – DynamoDB Table Creation (Basic)

## Simple Definition
Amazon DynamoDB is a fully managed, serverless NoSQL key-value database service.

## Objective
To create a DynamoDB table and insert sample items to understand serverless database basics.

## AWS Service Used
- Amazon DynamoDB

## Steps Performed
1. Created a DynamoDB table with partition key.
2. Used default table settings.
3. Inserted sample item using AWS console.
4. Verified inserted data in table items.

## Result
Successfully created a DynamoDB table and added sample items using AWS GUI.
