# Task 16 – RDS Database and SQL Workbench Connection

## Simple Definition
Amazon RDS is a managed relational database service that allows easy setup, operation, and scaling of databases.

## Objective
To create an RDS MySQL database and connect it using MySQL Workbench to execute SQL queries.

## AWS Services Used
- Amazon RDS (MySQL)
- Amazon EC2 (Security Group)
- MySQL Workbench

## Steps Performed
1. Created an RDS MySQL database using Free Tier.
2. Enabled public access and configured security group.
3. Connected RDS database using MySQL Workbench.
4. Created table and inserted data using SQL queries.

## Result
Successfully connected to RDS database and executed SQL queries using MySQL Workbench.
