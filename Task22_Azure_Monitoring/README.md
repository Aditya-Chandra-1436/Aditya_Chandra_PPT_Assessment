# Task 22 – Azure Monitoring & Diagnostics (Easy)

## Simple Definition
Azure Monitor is a service that collects and displays metrics and diagnostic data from Azure resources.

## Objective
To view monitoring metrics and diagnostics for Azure resources.

## Azure Service Used
- Azure Monitor
- Azure Storage Account

## Steps Performed
1. Opened an Azure Storage Account.
2. Viewed metrics using Azure Monitor.
3. Checked diagnostic settings.

## Result
Successfully viewed monitoring and diagnostic data without using a virtual machine.
