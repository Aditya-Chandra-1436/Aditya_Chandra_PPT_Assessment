# Task 19 – Azure Storage Account (Blob, File, Table)

## Simple Definition
Azure Storage provides scalable cloud storage services including Blob, File, and Table storage.

## Objective
To create an Azure storage account and explore Blob, File, and Table storage services.

## Azure Services Used
- Azure Storage Account
- Blob Storage
- File Storage
- Table Storage

## Steps Performed

2. Created a blob container and uploaded sample file.
3. Created a file share.
4. Created a table and inserted sample entity.

## Result
Successfully explored multiple Azure storage services within a single storage account.
