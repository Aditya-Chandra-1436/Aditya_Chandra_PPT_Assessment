# Task 8 – IAM Users and Groups

## Objective
To create IAM users and groups in AWS and manage permissions using group-based policies.

---

## Services Used
- AWS Identity and Access Management (IAM)

---

## Steps Performed
1. Opened IAM service from the AWS Management Console.
2. Created an IAM group named `Task8-Developers-Group`.
3. Attached the `AmazonS3ReadOnlyAccess` policy to the group.
4. Created an IAM user named `task8-user`.
5. Enabled AWS Management Console access for the user.
6. Added the user to the IAM group.
7. Saved user login credentials.
8. Verified successful login using the IAM user credentials.

---

## Outcome
IAM users and groups were successfully created, and permissions were managed using group-based policies.
