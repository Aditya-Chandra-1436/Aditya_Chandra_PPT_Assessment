# Task 15 – CloudFront Distribution for S3 Static Website

## Simple Definition
CloudFront is a Content Delivery Network (CDN) that delivers website content faster by caching it at global edge locations.

## Objective
To learn global content distribution by integrating an S3 static website with CloudFront.

## AWS Services Used
- Amazon S3
- Amazon CloudFront

## Steps Performed
1. Created an S3 bucket and uploaded static website files.
2. Enabled static website hosting on the S3 bucket.
3. Configured public access using bucket policy.
4. Created a CloudFront distribution with S3 static website as origin.
5. Configured cache behavior and HTTPS redirection.
6. Accessed the website using CloudFront distribution domain name.

## Result
The static website is successfully delivered through CloudFront CDN, providing faster and globally distributed access.
