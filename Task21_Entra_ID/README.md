# Task 21 – Entra ID (Users, Roles, Basic Identity Management)

## Simple Definition
Microsoft Entra ID is an identity and access management service used to manage users and roles in Azure.

## Objective
To create users and assign roles using Azure Entra ID.

## Azure Service Used
- Microsoft Entra ID

## Steps Performed
1. Accessed Microsoft Entra ID from Azure Portal.
2. Created a new user.
3. Assigned a Reader role to the user.
4. Verified role assignment.

## Result
Successfully managed identities by creating users and assigning roles in Entra ID.
