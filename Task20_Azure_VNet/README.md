# Task 20 – Azure VNet (Basic Networking Practical)

## Simple Definition
Azure Virtual Network (VNet) is a logically isolated network that enables secure communication between Azure resources.

## Objective
To create a virtual network, subnet, and configure basic NSG rules in Azure.

## Azure Services Used
- Azure Virtual Network
- Network Security Group (NSG)

## Steps Performed
1. Created a resource group.
2. Created a Virtual Network with custom address space.
3. Created a subnet within the VNet.
4. Configured Network Security Group with inbound SSH rule.
5. Verified subnet and NSG association.

## Result
Successfully created and configured an Azure Virtual Network with subnet and security rules.
