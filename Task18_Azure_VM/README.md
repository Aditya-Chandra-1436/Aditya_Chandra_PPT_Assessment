# Task 18 – Azure Virtual Machine (Easy Task)

## Simple Definition
An Azure Virtual Machine is a scalable compute resource that allows users to run applications in the cloud.

## Objective
To launch and configure a basic Azure virtual machine and access it using SSH.

## Cloud Platform Used
- Microsoft Azure

## Steps Performed
1. Created a resource group.
2. Launched an Ubuntu Linux virtual machine.
3. Configured networking and SSH access.
4. Connected to the VM using SSH.

## Result
Successfully created and accessed an Azure virtual machine using the Azure Portal.
